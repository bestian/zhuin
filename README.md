# 維基百科加注音

## 主要功能

這是一個教育性的擴充功能，可以讓您點擊擴充功能的小圖示以切換成注音模式。

注音模式會讓維基百科的字體變大，並且加上注音，方便小朋友使用。

## 安裝步驟

1. **下載「源樣注音明體」**
   [下載連結](https://github.com/ButTaiwan/bpmfvs/releases/download/v1.320/BpmfGenYoGothic.zip)

2. 將壓縮檔解壓縮，進入解壓後的資料夾

3. 點擊 "BpmfGenYoGothic-R.ttf" 以安裝「源樣注音明體」

4. 安裝完字體後，接著安裝此擴充功能

### Chrome 擴充功能商店連結：

[維基百科加注音](https://chromewebstore.google.com/detail/%E7%B6%AD%E5%9F%BA%E7%99%BE%E7%A7%91%E5%8A%A0%E6%B3%A8%E9%9F%B3/bklcpdppephbnjdfpofichhacblhbknf?hl=zh-TW)

## 使用方法

安裝完成後，點擊瀏覽器右上角的擴充功能圖示，即可切換注音模式。
